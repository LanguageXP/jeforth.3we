[ ] Words(a[]) <--- why a[] ?
    A: 這是很老的 code 了。把 word name, xt, 等一切 properties 用一個 array 送進去。
       這個機關現在看來有點多餘， JavaScript object 的 properties 可以隨時加掛，不需要用
       這樣的技巧。位置在 project-k 的 projectk.js 裡面，誰願意貢獻時間有空把它改掉。
[ ] panic() <---- why not simply type() ?
    A: jeforth engine or kernel is supposed to be used in embeded systems too.
       In that case, panic() is still necessary while type() has no meaning.
